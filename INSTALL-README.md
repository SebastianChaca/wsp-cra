# Whatsapp Web Clone installation guide

# Frontend
1. Cd to the frontend folder
2. Install project:  

```
npm i
```
3. Create `.env` file using `.env.template` as guide
3. Start project
```
npm start
```

# Backend
1. cd to backend/v1
2. install project 
```
npm i
```
3. Create `.env` file using `.env.template` as guide
4. Data base 
 ```
   docker compose up -d
   ```
5. Start project
```
npm run start:dev
```
