# Whatsapp Web Clone installation guide

# Frontend
1. install project 
```
npm i
```
2. Create `.env` file using `.env.template` as guide
3. Start project
```
npm start
```

# Backend
1. install project 
```
npm i
```
2. Create `.env` file using `.env.template` as guide
3. Data base 
 ```
   docker compose up -d
   ```
4. Start project
```
npm start
```
